# 视觉规范

项目所有原型页面的统一视觉规范入口。规范决策优先级见 `design-guide.md`，本文档讲具体查找、应用与维护步骤。

## 规范文件位置

视觉规范统一存放在 `src/themes/visual-spec.md`。

- **存在**：直接读取，严格遵循其中定义的配色、字体、间距、组件样式与交互风格。
- **不存在**：引导用户使用 `extract-workflow` 技能从 Axure / Figma / 参考网页中提取并落盘。

完成首个原型后，建议提醒用户将本次使用的视觉风格整理到 `src/themes/visual-spec.md`，供后续原型复用。

## 维护规则

- 新增前先检查 `src/themes/visual-spec.md` 是否已有同类定义，能复用就不新增
- 未经用户确认，不删除、不覆盖已有视觉规范内容
