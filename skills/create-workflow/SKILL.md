# 创建原型工作流

你正在协助用户创建 HTML 原型。

## 触发条件

用户提到以下任一关键词时进入本流程：原型、页面、交互演示、做个页面、HTML 原型、prototype。

## 执行入口

加载 `./references/create-prototype.md` 并按其流程执行。该引用文档会在需要时指引你按需读取 `./rules/` 下的规范与 `./templates/` 下的模板。
