# 创建原型工作流

你正在协助用户创建 HTML 原型。

## 触发条件

用户提到以下任一关键词时进入本流程：原型、页面、交互演示、做个页面、HTML 原型、静态原型、走查稿、prototype。

## 分流

收到创建请求后，先判断原型类型：

- 用户明确说"静态""走查""只看布局""不要交互""平铺" → 加载 `./references/create-static-prototype.md`
- 用户明确说"交互""可点击""演示流程""有状态切换" → 加载 `./references/create-prototype.md`
- 未说明 → 使用以下模板询问，等待用户回复后再加载对应文档：

```text
请问需要哪种类型？

1. 交互原型（可点击，有状态切换与交互逻辑）
2. 静态原型（所有页面与弹窗平铺展示，无交互逻辑，适合走查与评审）
```
